# Electric Vehicle Population Data Analysis

## Project Overview
This project focuses on analyzing Electric Vehicle (EV) population data using MongoDB. It helps in understanding EV registration trends and managing large-scale datasets efficiently.

## Key Features
- Designed a scalable database schema for electric vehicle registration data  
- Efficient data storage and retrieval using MongoDB  
- Optimized indexing and data modeling for better performance  
- Improved query execution time by 31%  
- Data analysis for EV population trends and insights  

## Technologies Used
- MongoDB  
- Data Modeling    
- Database Management  

## Project Impact
This project demonstrates strong database management and optimization skills. It focuses on handling large datasets efficiently and improving system performance for real-world analytics use cases.
